# Ejercicio 1 — Clasificación Multiclase (Dígitos)

En este ejercicio se trabaja un problema de clasificación multiclase
utilizando imágenes de dígitos escritos a mano (0–9) representadas
como vectores de características.

## Objetivos
- Comprender clasificación multiclase
- Usar Classification Learner de MATLAB
- Analizar matrices de confusión 10×10
- Evaluar impacto de hiperparámetros

## Dataset
digits_multiclass.csv  
Clases: 0–9  
Variables: 64

## Actividades
- Entrenar KNN variando K
- Comparar SVM y Árbol
- Analizar errores entre clases similares (3 vs 8, 4 vs 9)

## Nota
Para demostraciones en clase se usa una versión reducida del dataset.
