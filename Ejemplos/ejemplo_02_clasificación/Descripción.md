# Ejemplo 2 — Clasificación con Classification Learner

En este ejemplo se muestra el flujo completo de entrenamiento de modelos de
clasificación utilizando la herramienta gráfica Classification Learner de MATLAB.

## Objetivos
- Preparar los datos para clasificación
- Entrenar y comparar modelos
- Ajustar hiperparámetros
- Analizar métricas de desempeño
- Interpretar la matriz de confusión

## Dataset
Credit Card Fraud Dataset (descargado externamente)

## Nota
El entrenamiento se realiza desde la interfaz gráfica de MATLAB.

## ⚠️ Nota sobre desempeño:
Para demostraciones en vivo se utiliza una versión reducida del dataset
con el fin de acelerar el entrenamiento. El dataset completo se deja
como material opcional para ejecución autonoma.
