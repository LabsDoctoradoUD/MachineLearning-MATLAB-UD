# Ejemplo 3 — Regresión con Regression Learner

Este ejemplo demuestra el flujo completo de un problema de regresión
utilizando el Energy Efficiency Dataset y la herramienta gráfica
Regression Learner de MATLAB.

## Objetivos
- Preparar datos para regresión
- Entrenar y comparar modelos
- Interpretar métricas RMSE y R²
- Analizar generalización vs complejidad

## Dataset
Energy Efficiency Dataset (Heating Load - Y1)

## ⚠️ Nota sobre desempeño:
Para demostraciones en vivo se utiliza una versión reducida del dataset
activando USE_DEMO_DATA = true.
