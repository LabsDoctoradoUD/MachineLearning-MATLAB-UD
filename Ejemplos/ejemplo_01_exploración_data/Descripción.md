# Ejemplo 1 — Exploración del problema de Machine Learning

Este ejemplo muestra cómo analizar un dataset real para identificar:
- el tipo de problema,
- la variable objetivo,
- la distribución de los datos,
antes de entrenar cualquier modelo.


## Dataset: Credit Card Fraud
## Objetivo: Identificar el tipo de problema y entender los datos


## Discusión guiada

- 1. ¿Este es un problema de clasificación o regresión?

- 3. ¿Es suficiente la accuracy como métrica?
