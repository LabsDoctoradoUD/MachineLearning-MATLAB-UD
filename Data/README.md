## Dataset de Clasificación — Credit Card Fraud

Este dataset no se incluye directamente en el repositorio debido a su tamaño.

### Pasos para descargar:

1. Ir a:
   https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

2. Descargar el archivo `creditcard.csv`

3. Copiar el archivo descargado dentro de esta carpeta:

   MachineLearning-MATLAB-UD/data/

4. Verificar que el archivo se llame exactamente:

   creditcard.csv

Una vez ubicado el archivo en esta carpeta, los scripts del curso funcionarán correctamente.
